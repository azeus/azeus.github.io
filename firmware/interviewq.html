Embedded Basics

  • Loop Count down vs Count up

    • Count down is always better because in assembly the is an instruction to compare with zero so there will not be any need to load another number for comparision when we count up.

  • What are the Data types supportd in C?

    • There are three kinds of Data types supported in C : Basic, Derived and User Defined. Integer, char, float and void are basic, Arrays, pointers and functions are derived and struct, union, enum and typedef are user defined

    Computer Architecture

  • Big vs Little Endianness

    • In big endian way of byte addressing the MSB is placed on the lower address and in lil endian way of byte addresssing the LSB is placed on the lowest address.
